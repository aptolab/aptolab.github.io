Bienvenido a Aptolab.

El rol de Aptolab es construir software y hardware inteligentes para uso cotidianos, asi como intecraccion humano-computadora , bases de conocimiento y motores de busqueda personalizados.

Si te es familiar alguno de estos termino seguro tienes buenos conocimientos, de momento estamos desarrollando nuestro bunker de aplicaciones para el publico y las podras encontrar aqui.

Servidores Remotos

Las aplicaciones que funcionan como servidores remotos, los escribimos en codigo Python o C#, y optamos por dar actuacion tanto pasiva como activa para dar soluciones a problemas que se generen en diferentes entornos.

Bases de Datos

Utilizamos bases de datos de caracter estanr SQL como mysql o sqlite y tambien usamos bases de datos NO-SQL como mongodb para tener una mayor escalabilidad del las aplicaciones que desarrollamos.
