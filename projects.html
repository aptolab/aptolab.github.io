A continuacion presentamos los proyectos en los que estamos trabajando actualmente.

Atiruz

Atiruz es un software enfocado a la asistencia personal y control de dispositivos locales y remotos por comandos de voz personalizados.Cuenta con un framework de desarrollo para extender sus capacidades y con un sistema de desarrollo de plugins para adaptarlo a diferentes aplicaciones.

Cuenta con un motor de Inteligencia Artificial para asistirnos y darnos ayudas sobre las tareas que tenga encomendadas, ademas mantiene los recuerdos de las ultimas tareas procesadas y genera reportes de tendencias y generadores logicos para tratar de extender las tareas que se ejecutaran.


  • S.O Windows , Android
  • Escrito en C#
  • Documentacion
  • Leer Mas

Apocaliptix

Apocaliptix es una aplicacion para el manejo y documentacion de proyectos que se enfoca principalmente en la movilidad, es una aplicacion movil que intenta llevar el control de las anotaciones,imagenes,audios que se nos ocurran y tiene la posibilidad de sincronizarze con un servidor de respaldo en PC.


  • S.O Android
  • Escrito en Java
  • Documentacion
  • Leer Mas

Fenderix

Es una libreria escrita en java que sirve para hacer operaciones matematicas sobre los mathplace, ayuda principalmente a la hora de llevar un control del historial de las operaciones, asi como poder modificar formulas y obtener resultados entiempo real, ademas la posibilidad de exportar los mathplaces. para compartirlos con amigos o manipularlos en PC.


  • S.O Android
  • Escrito en Java
